# Small Changes, Big Impacts

Try the demos on this page to get a sense for what you'll learn in this course. In each one, you can
make small changes using the input, see the change that makes to the code, and see a big impact in
the final product.

## Change the Text

Change the text to see how the code and output change.

Try to come up with a better title and tagline for the site.

<MagicDemo
  initialState={{title: 'Intro to Recycling', tagline: 'Learn about recycling'}}
  input={(state, updateState) => (
    <>
      Title:{' '}
      <input value={state.title} onChange={e => updateState({...state, title: e.target.value})} />
      Tagline: <input
        value={state.tagline}
        onChange={e => updateState({...state, tagline: e.target.value})}
      />
    </>
  )}
  template={state => `<div>
  <h1>${state.title}</h1>
  <h2>${state.tagline}</h2>
</div>`}
/>

## Change the Colors

Choose the color palette for the site. Pick a background color and a text color.

<MagicDemo
  initialState={{backgroundColor: '#EEEEEE', color: '#111111'}}
  input={(state, updateState) => (
    <>
      Background Color:{' '}
      <input
        type="color"
        value={state.backgroundColor}
        onInput={e => updateState({...state, backgroundColor: e.target.value})}
      />
      Text Color: <input type="color" value={state.color} onInput={e => updateState({...state, color: e.target.value})} />
    </>
  )}
  template={state =>
    `<style>
  div {
    background-color: ${state.backgroundColor};
  }
  h1, h2 {
    color: ${state.color};
  }
</style>
<div>
  <h1>Small Changes, Big Impacts</h1>
  <h2>An Exploration of Climate Action</h2>
</div>`
  }
/>

## Changing which element moves

Choose which elements should be animated

<MagicDemo
  initialState={['h2']}
  input={(state, updateState) => (
    <>
      <label for="h1">The title</label>
      <input
        type="checkbox"
        id="h1"
        defaultChecked={state.includes('h1')}
        onChange={e =>
          updateState(e.target.checked ? [...state, 'h1'] : state.filter(el => el !== 'h1'))
        }
      />
      <label for="h2">The tagline</label>
      <input
        type="checkbox"
        id="h2"
        defaultChecked={state.includes('h2')}
        onChange={e =>
          updateState(e.target.checked ? [...state, 'h2'] : state.filter(el => el !== 'h2'))
        }
      />
    </>
  )}
  template={state => `<style>
@keyframes slideInFromLeft {
  0% {
      transform: translateX(-100%);
  }
  100% {
      transform: translateX(0);
  }
}
${state.join(', ')} {
  animation: 1s ease-out 0s 1 slideInFromLeft;
}
</style>
<div>
  <h1>Small Changes, Big Impacts</h1>
  <h2>An Exploration of Climate Action</h2>
</div>`}
/>

## Choose how many elements to show

<MagicDemo
  initialState={{count: 15}}
  input={(state, updateState) => (
    <>
      How many to show:{' '}
      <input
        type="number"
        value={state.count}
        onChange={e => updateState({count: e.target.value})}
      />
    </>
  )}
  template={state => `
<h1>Big list of impacts you can have</h1>
<ol></ol>
<script>
  var list = document.querySelector('ol')
  list.innerHTML = '';
  for (let i = 0; i < ${state.count}; i++) {
    list.innerHTML += '<li>Another one</li>'
  }
</script>
`}
/>
